Configure Hostnames
    Change Hostname
        $ sudo -i 
        $ nano /etc/hostname  
            - change the name to: mail.<sitename>.com 
    Edit Host File 
        $ nano /etc/hosts 
            - add: 127.0.0.1  mail.<sitename>.com  localhost 
        $ hostname mail.<sitename>.com 
        $ hostname -f - it should output the hostname
