Install and enable Simpletest
Testing using GUI: configuration -> Testing 

Testing using drush: 
$ drush | grep test 
$ drush help test-run //to see avialable command options
$ drush test-run //show all the test case that you can run 
$ drush st //to see the drush site 
$ drush - l <url> test-run <test-case>
